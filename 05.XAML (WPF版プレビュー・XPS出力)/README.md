# XAML (WPF版プレビュー・XPS出力) サンプル

このサンプルでは、WPF の XAML を使った **印刷プレビュー** と **XPS ファイル出力** を実演します。

---

## 概要

- **XAML ベースのプレビュー**: WPF の DocumentViewer を使い、帳票をプレビュー表示します。
- **XPS 出力**: プレビュー画面から XPS 形式でファイル出力が可能です。
- **カスタムXAML**: `Custom.xaml` でプレビューの外観をカスタマイズできます。

---

## スタートアップ プロジェクトの切り替え

このサンプルには **C#** と **VB** の両方のプロジェクトが含まれています。
ソリューションエクスプローラーで目的のプロジェクトを右クリックし、
**「スタートアップ プロジェクトに設定(S)」** を選択してください。
